#Mode switch explanation

mode Timeout = 10 secs

mode Normal
-----------
Pa (short) - Change display of Temp and Humidity
Pa (long) - Change to mode Select interval
Pb (short) - Change to data view mode.
Pb (long) - Reset data storage.

mode Select Interval
--------------------
Pa (short) - select storage interval in minutes.

mode Data View:
---------------
Pa (short) - moves to the latest reading subsequent pushes ++
Pa (short) - moves to the oldest reading subsequent pushes --
